360t Billing Interface

  • Einleitung

This console is a GUI interface for 360T billing service that handles billing calculations for the company customers. The interface accomodates to the complex relation ships between the different modules in that billing service by representing them in an entity diagrams that are maintainable and easy to view, this will facilitates the process to a great deal.

The IDE tools is cross platform, built using SMYL special graphics components, like the dockable internal frames that the user can drag and drop them on the sides of the screen as well as minimizing them into icons on the lower bar of the MDI window to give more working space for other active windows.Dockability is also reflected on the usual window components, for example the user can drag and drop the tables list onto the different sides of the window.The console supports multilinguality also, taking into consideration the component orientation related to each language.
